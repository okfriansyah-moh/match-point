About Match Point

Product documentation for players, club admins, and platform operators — goals, features, rules, and release notes.

Vision & goals

Match Point is a cross-community reputation and ranking platform for padel, tennis, pickleball, badminton, and table tennis. Most play happens outside official federations — recreational leagues, open play, and club events leave no portable record. We fix that with verified identity, match results, and rankings that travel with the player.

GoalWhat it means for users
Portability Your rank, endorsements, and community history follow you across clubs and cities.
Trust Organizer-submitted results, phone verification, dispute flow, and transparent rating rules.
Community-first Clubs run day-to-day play; the platform handles cross-community seasons and global tiers.
Fair play Skill brackets, anti-sandbagging gates, and public rules — no hidden rating logic.

Communities

A community is a club or group (open or invite-only) tied to one primary sport. Players can belong to multiple communities with one home community on their profile.

  • Discovery feed — find clubs by sport, distance, and access type
  • Community page — public spotlight, member feed, join CTA
  • Platform approval for new communities and featured listings
  • Admin dashboard — members, pending reviews, matches, active events

Rankings

One ranking engine per sport — three numbers on your profile:

LayerNamePurpose
1 MP Rating Trusted skill from verified matches — brackets, seeding, anti-sandbagging
2 MP pts (Mabar / Global) Club leaderboard & cross-community prestige — starts at 0
3 Journey tier Pemula → Berkembang → Mapan — from ladder pts + stability

Which tournament should I join?

  1. No verified match yet → Open only, or play your first match
  2. Provisional (1–4 matches) → Beginner or Open
  3. Stable → compare your MP Rating to the bracket range on the event card

Bracket Class = who can register. Global Tier 1–3 = how many Global pts the winner earns.

Global tournaments

Platform admin runs cross-community tournaments (Tier 1–3) that feed Global pts. Club tournaments use the same bracket classes with MP Rating gates.

Battle of Communities (BoC)

Flagship seasonal inter-community product — Match Point branded only (not third-party league co-branding). Operators with 100+ communities use in-app draw instead of external randomizers; squads and penalties are tracked in-platform.

AspectRule
Who runs itPlatform admin
ScaleMany communities (10–100+), group draw → round-robin → knockout
Rubric3 doubles + 1 singles war points; optional King/Queen (2 pts)
Scoring8 games per set (configurable)
SquadsDue 3 days before each fixture; late → −1 game per set
Rank weight100% of a standard Mabar fixture delta
AuditDraw seed, operator, timestamp logged immutably

Ratings & fair play

MP Rating (Glicko-2-lite) per sport: one trusted number from verified matches. Reference hints (≈ WPR, ≈ UTR) are optional — never the gate.

Is MP Rating fair across clubs?

If Club A is stronger than Club B and both only play internally, MP Rating 4.0 at Club A is not promised to equal 4.0 at Club B — yet.

ScopeBadgeFair for
Club-local Klub / Club Same-club brackets, Mabar leaderboard
Cross-community Lintas klub ✓ Global tournaments, BoC, inter-club sparring

When clubs meet, verified results calibrate both sides via Glicko expected-outcome — upsets move ratings fairly. Run inter-club sparring to earn Lintas klub ✓.

Anti-sandbagging

  • Registration: min/max MP Rating per bracket class
  • Play-up allowed when configured; play-down blocked by default
  • Global / inter-club Intermediate+ requires Lintas klub ✓ calibration
  • BoC squad slots re-check eligibility before lock

Sports & scale

The same ranking rules apply to all five sports — padel, tennis, pickleball, badminton, and table tennis.

  • One identity, separate MP Rating and ladder per sport — no cross-sport bleed
  • Same bracket vocabulary (Beginner / Intermediate / …) — only numeric scale differs
  • Adding a sport = new scale calibration in SPORT_PROFILES, not new rules
